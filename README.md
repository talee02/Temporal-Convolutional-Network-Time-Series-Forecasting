# Temporal-Convolutional-Network-Time-Series-Forecasting
## Project Dataset
This project was build using primary data from the Indonesian Food and Drug Administration (BPOM) for the period 2021-2024.
| **Variable** | **Data Type** | **Description** |
|---|---|---|
| Pharmaceutical Wholesaler | String | Name of the Pharmaceutical Wholesaler |
| Amount | Numerical | Distribution Amount per Day |
